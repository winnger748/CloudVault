# CloudVault — Облачное хранилище нового поколения

CloudVault — это современный лендинг для облачного сервиса хранения данных. Проект демонстрирует UI/UX уровня SaaS-продуктов с акцентом на безопасность, скорость и удобство использования.

## 📌 О проекте

Данный проект представляет собой одностраничный сайт (Landing Page), разработанный для презентации облачной платформы хранения данных.

Основная цель — визуально и функционально показать:

* преимущества сервиса
* ключевые возможности продукта
* тарифные планы
* сценарии использования

Лендинг выполнен в современном технологичном стиле с использованием glassmorphism и неоновых акцентов.

## ⚙️ Технологии

Проект реализован без использования фреймворков:

* HTML5
* CSS3 (Flexbox, Grid, кастомные переменные)
* Vanilla JavaScript

Без сторонних библиотек 

## 🎨 Особенности дизайна

* Тёмная тема с неоновыми элементами
* Glassmorphism интерфейс
* Адаптивная верстка (mobile-first)
* Плавный скролл и анимации
* Минималистичная, чистая структура

## 🚀 Функциональность

* Адаптивное меню (hamburger для мобильных устройств)
* Плавная навигация по якорям
* Интерактивные UI-элементы
* Гибкая сетка контента
* Структурированные секции:

  * Hero-блок
  * Возможности
  * Как это работает
  * Тарифы
  * Footer

## 📱 Адаптивность

Сайт корректно отображается на:

* десктопах
* планшетах
* мобильных устройствах

Используются media queries для оптимизации интерфейса под разные размеры экранов.

## 📂 Структура проекта

Проект состоит из одного файла:

```
index.html
```

Вся логика, стили и разметка находятся внутри одного файла для упрощённого деплоя и демонстрации.
